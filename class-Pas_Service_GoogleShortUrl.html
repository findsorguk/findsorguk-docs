Class Pas_Service_GoogleShortUrl

A service class for shortening and expanding links with goo.gl

An example of use:

<?php
$shortener = new Pas_Service_GoogleShortUrl($key);
$url = $shortener->expand($url);
$analytics = $shortener->analytics($url);
?>
