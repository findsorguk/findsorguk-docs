Class Pas_Geo_Gridcalc

A script for manipulating grid references Primarily for converting OSGB to LatLon. This uses some code from nearby.org.uk, but has been heavily adapted. The UTM functions are adapted from https://gist.github.com/840476

An example of use:

<?php
$conversion = new Pas_Geo_Gridcalc($data['gridref']);
$results = $conversion->convert();
?>
