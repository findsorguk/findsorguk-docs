Class Pas_Geo_BoundBox

A class for creating a new latitude/longitude pair at a distance from a point. Bearing is in degrees

Package: Pas\Geo
Category: Pas
License: GNU Affero GPL v3.0
Author:

Daniel Pett dpett@britishmuseum.org

Example usage:

<?php
 $geo = new Pas_Geo_BoundBox();
 $newPoint = $geo->createPoint(53.1,-2.0, 45, null,true);
 If creating a square use bearings of 45, 135, 225 and 315 degrees
?>
