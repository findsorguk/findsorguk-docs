Class Pas_View_Helper_AuditLogs

A view helper for rendering audit logs on a finds record

This helper uses data from the finds, findspots and coins models and then redisplays this as a list of audited actions that are clickable and this instatiates a modal window with the changes within it.

To use this, follow the example below:

<?php
echo $this->auditLogs()->setID(1);
?>
