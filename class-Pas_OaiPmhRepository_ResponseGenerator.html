Class Pas_OaiPmhRepository_ResponseGenerator

Modified in places for the creation of item data via solr OaiPmhRepository_ResponseGenerator generates the XML responses to OAI-PMH requests recieved by the repository. The DOM extension is used to generate all the XML output on-the-fly.
