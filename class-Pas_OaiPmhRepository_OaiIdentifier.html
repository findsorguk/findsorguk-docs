Class Pas_OaiPmhRepository_OaiIdentifier

Utility class for dealing with OAI identifiers

OaiPmhRepository_OaiIdentifier represents an instance of a unique identifier for the repository conforming to the oai-identifier recommendation. The class can parse the local ID out of a given identifier string, or create a new identifier by specifing the local ID of the item.
