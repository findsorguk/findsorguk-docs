Class Pas_View_Helper_ResultsSorter

A view helper for setting up a results sorter

This helper is used on any page where results are taken from the solr index and then offers the user the chance to sort. It could be set up, so that the user could have a default sort column and order in their preferences (one day!) Example of use: resultsSorter()->setResults($this->paginator); ?>
