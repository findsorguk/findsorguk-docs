Class Pas_Geo_Edina_NameAndFeatureSearch

An interface to the Edina NameAndFeatureSearch api call

Pas_Geo_Edina
Pas_Geo_Edina_NameAndFeatureSearch
Package: Pas\Geo\Edina\NameAndFeatureSearch
Category: Pas
Copyright: Daniel Pett, The British Museum
License: GNU Affero GPL v3.0
Author: Daniel Pett
Version: 1
Since: 3/2/12
See:

http://unlock.edina.ac.uk/places/queries/ Usage:

$edina = new Pas_Geo_Edina_NameAndFeatureSearch(); $edina->setFeatureType('Cities'); $edina->setNames(array('cambridge')); $edina->setFormat('json'); - you can use georss

, kml, xml,

jaon $edina->setGazetteer('geonames'); - you can use unlock

, os,

geonames $edina->get(); You can also get the names and features queried back from: $edina->getFeatureTypes(); $edina->getNames(); If you want to get the url of the api call $edina->getUrl();

Then process the object returned as you want (up to you!)
