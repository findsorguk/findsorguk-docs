Class Pas_Calendar_Mapper

This class is a mapping gateway to retrieving data from the Google Calendar An example of code use:

<?php
$this->_gcal = new Pas_Calendar_Mapper();
$event = $this->_gcal->getEvent($this->_getParam('id'));
?>
